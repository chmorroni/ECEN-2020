CAR:

Debugging:
	Logging
		Received packets
		Sensor values
		Adjustments made for safety
		NRF status
		Timestaps
	Commands
		v: Toggle verbose output
		i: Info (sensors, motors, turning, last command, NRF)
		s: Speed
			- Wait for number (speed)
			- Wait for number (repeats)
		t: Turn
			- Wait for direction {L, R, S}
		e: Enable
		l: Toggle lights
		d: Disable

		Setup NRF
	Catch exceptions (?)

		Blink lights
	Reading sensors
		Timer (20Hz)

		2 light sensors
	Command processing
		Interrupt from NRF
			Decode into struct
			Safetify
				Preserve direction

Drivers:
